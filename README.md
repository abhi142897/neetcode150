# NeetCode 150 — Java Notes

Topic-wise interview prep notes for the [NeetCode 150](https://neetcode.io/practice) problem set, written in Java.

## Why this repo exists

I'm using this repo as a single source of truth while prepping for coding interviews. Each topic gets one markdown file that covers:

- The underlying data structure/concept: how it works, how to visualize it, traversal and time complexity, comparisons with similar structures, and when to use it.
- Every problem in that topic: explanation, intuition, key gotchas, brute-force approach, and the optimal approach with Java code and complexity analysis.

The goal is a file I can skim in 10–15 minutes the night before an interview and walk away with the core patterns fresh in my head.

## How to use these notes

Each topic file follows the same template:

1. **Data structure primer** — definition, types, visualization, traversal, time complexity, pros/cons, comparison with alternatives.
2. **Problem-by-problem breakdown** — for every problem: explanation → intuition → things to remember → brute force → optimal approach (with code + complexity).
